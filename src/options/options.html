How to get a YouTube Data API key

  1. Go to the Google Cloud Console and create or select a project.
  2. Open APIs & Services → Library and search for "YouTube Data API v3". Click Enable.
  3. Open APIs & Services → Credentials, click Create Credentials → API key.
  4. (Recommended) Restrict the key to the YouTube Data API v3 and, optionally, to browser extension HTTP referrers.
  5. Copy the key and paste it above.

Default quota is 10,000 units/day. A full scan of ~270 channels costs roughly 280 units, so you get ~35 full scans per day before hitting the limit.
